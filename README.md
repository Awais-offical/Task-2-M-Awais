DecodeLabs Data Analytics Project
Project Overview
This project focuses on analyzing an e-commerce sales dataset to identify patterns, trends, and distributions using Python in Google Colab. The analysis includes descriptive statistics, visualization, trend identification, and outlier detection to gain meaningful business insights.

Objectives

- Analyze the dataset using descriptive statistics.
- Calculate mean, median, and count.
- Identify sales trends.
- Detect outliers.
- Visualize data using charts.
- Summarize key findings.

Dataset Information

The dataset contains **1,200 records** and **14 columns** related to e-commerce orders.
 Columns
